# 🧃 OWASP Juice Shop - TryHackMe Writeup & Pentest Report

Welcome to my repository for the [OWASP Juice Shop](https://tryhackme.com/room/owaspjuiceshop) room on **TryHackMe**.

This repository serves as a comprehensive collection of my penetration testing reports, vulnerability assessments, and step-by-step writeups for the challenges encountered in the Juice Shop application. OWASP Juice Shop is arguably the most modern and sophisticated insecure web application, used widely for security training, awareness, and testing tools.

## 🎯 Room Overview

- **Platform**: [TryHackMe](https://tryhackme.com/)
- **Room**: [OWASP Juice Shop](https://tryhackme.com/room/owaspjuiceshop)
- **Difficulty**: Varies (Trivial to Extreme)
- **Focus**: Web Application Penetration Testing, OWASP Top 10

## 📂 Repository Contents

This repository is organized to provide clear and structured documentation of my findings and exploitation methods:

- 📝 **Penetration Test Reports**: Comprehensive, professional-grade reports summarizing the scope, methodology, executive summary, and detailed findings of the assessment (`.md`, `.pdf`).
- 🐛 **Vulnerability Assessments**: Specific documents outlining technical details, impact, and remediation for the vulnerabilities discovered (`.docx`, `.pdf`).
- 📖 **Challenge Writeups**: Detailed, step-by-step Markdown and PDF guides explaining the exploitation process for various Juice Shop challenges.
- 📓 **Assessment Notes**: Raw, unedited notes taken during the active testing phase.
- 📸 **Screenshots**: A dedicated folder containing visual evidence and proof-of-concepts (PoCs) captured during the testing process.
